# Portfolio PT. Hitech Aksara Digital

React + Vite + Tailwind CSS website based on the portfolio design.

## Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

## Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
  components/
    Navbar.jsx
    Hero.jsx
    ClientLogoCloud.jsx
    CaseStudyGallery.jsx
    Testimonial.jsx
    SuccessMetrics.jsx
    Footer.jsx
  App.jsx
  main.jsx
  index.css
```

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Material Symbols Outlined
